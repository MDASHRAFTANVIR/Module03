Let's know more about the difference between HTML4 and HTML5

Feature HTML4 HTML5
DOCTYPE declaration < !DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd" > < !DOCTYPE html >
Semantic tags Limited support for semantic tags Introduced several new semantic tags (e.g. < header > , < footer > , < nav > )
Audio and video support Required third-party plugins like Flash or QuickTime Native support for audio and video playback using the < audio > and < video > tags
Form controls Limited form control options Introduced several new form controls (e.g. date pickers, sliders, color pickers)
Graphics technologies Limited graphics capabilities Introduced Canvas and SVG, providing more advanced graphics capabilities
Offline and storage capabilities No support for offline or local storage Introduced APIs for offline storage and local data storage, supporting web applications that continue to function even when there is no internet connection available
